Color Systems in HTML and CSS

Color systems in HTML and CSS allow you to specify colors for elements in different ways. The most common are color names, RGB values, and Hexadecimal values.

1. Color System Names

HTML and CSS support a set of predefined color names, like red, blue, green, orange, etc. These are easy to remember and use:

This text is red.

This background is light blue.

  • There are 140+ standard color names.
  • They are case-insensitive (Red is the same as red).

2. RGB (Red, Green, Blue)

RGB stands for Red, Green, Blue. You define a color by specifying the intensity of each component:

This text is red.

This background is light blue.

  • Each value ranges from 0 to 255.
  • rgb(0, 0, 0) is black, rgb(255, 255, 255) is white.
  • You can also use rgba() to add an alpha (opacity) value.

3. Hexadecimal Colors

Hexadecimal (hex) colors use a #RRGGBB format, where RR, GG, and BB are two-digit hexadecimal numbers (00 to FF) representing red, green, and blue:

This text is red.

This background is light blue.

  • Hex values start with # and are followed by 3 or 6 hexadecimal digits.
  • #000000 is black, #ffffff is white.
  • Short form: #f00 is the same as #ff0000.

Gotchas

  • Not all browsers render named colors exactly the same, but differences are rare.
  • RGB and Hex values give you more control and allow for millions of colors, while names are limited.
