Including CSS

This page demonstrates how to include CSS in an HTML document.

CSS can be included in three ways: inline, internal, and external.

In this example, we are using an external stylesheet linked in the head section of the HTML document.

Check the styles.css file for the styles applied to this page.

Using external stylesheets helps keep your HTML clean and separates content from presentation.

Feel free to modify the styles.css file to see how it affects the styling of this page.

Remember, using external stylesheets is a best practice for larger projects as it allows for better organization and reusability of styles across multiple pages.

Happy styling!

External Styles

External styles are defined in a separate CSS file and linked to your HTML document using the <link> tag in the <head> section. This approach keeps your HTML clean and allows you to reuse the same styles across multiple pages.

For example, the styles.css file linked above contains styles that are applied to this page. You can edit styles.css to see how external styles affect the appearance of your HTML elements.

Using external stylesheets is a best practice for larger projects as it allows for better organization and reusability of styles across multiple pages.

Happy styling!
