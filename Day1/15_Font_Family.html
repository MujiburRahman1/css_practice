CSS Font Family Examples

This is a serif font. Serif fonts have small lines at the ends of characters. Example: Times New Roman.

This is a sans-serif font. Sans-serif fonts do not have the small lines. Example: Arial.

This is a monospace font. All characters take up the same amount of space. Example: Courier New.

This is a cursive font. Cursive fonts look like handwriting. Example: Brush Script MT.

This is a fantasy font. Fantasy fonts are decorative. Example: Impact.

How it works:
The font-family property lets you list several fonts. The browser uses the first available one. The last value (like serif or sans-serif) is a generic family as a fallback.
