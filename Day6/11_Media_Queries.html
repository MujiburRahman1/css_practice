CSS Media Queries Complete Guide

Media queries allow you to apply CSS styles depending on device characteristics like width, height, orientation, and more.

1. Basic Syntax

@media media-type and (media-feature: value) { /* CSS rules */ }

2. Common Media Features

Width and Height

/* Applies when viewport is 600px or smaller */ @media (max-width: 600px) { /* styles */ } /* Applies when viewport is 600px or larger */ @media (min-width: 600px) { /* styles */ } /* Range between 600px and 900px */ @media (min-width: 600px) and (max-width: 900px) { /* styles */ }

Orientation

/* Portrait mode (height > width) */ @media (orientation: portrait) { /* styles */ } /* Landscape mode (width > height) */ @media (orientation: landscape) { /* styles */ }
Rotate your device to see orientation changes

Resolution

/* High resolution displays */ @media (min-resolution: 2dppx) { /* styles */ } /* Retina displays */ @media (-webkit-min-device-pixel-ratio: 2) { /* styles */ }

3. Device Type Examples

Mobile Devices (under 600px)

@media (max-width: 600px) { body { font-size: 14px; } }
This box turns red on screens smaller than 600px

Tablets (601px to 900px)

@media (min-width: 601px) and (max-width: 900px) { .sidebar { width: 40%; } }
This box turns green between 601px and 900px

Desktop (901px and above)

@media (min-width: 901px) { .main-content { display: grid; grid-template-columns: 2fr 1fr; } }

5. Print Styles

@media print { nav, footer { display: none; } body { font-size: 12pt; line-height: 1.5; } a::after { content: " (" attr(href) ")"; } }
This box will not appear when printed (try Ctrl+P)

6. User Preference Queries

Dark Mode

@media (prefers-color-scheme: dark) { body { background-color: #222; color: #eee; } }
This box changes in dark mode (check your OS settings)

Reduced Motion

@media (prefers-reduced-motion: reduce) { .animation { animation: none; transition: none; } }

7. Breakpoint Strategy

Common Breakpoints

/* Small devices (phones, 600px and down) */ @media (max-width: 600px) { ... } /* Medium devices (tablets, 601px to 900px) */ @media (min-width: 601px) and (max-width: 900px) { ... } /* Large devices (desktops, 901px and up) */ @media (min-width: 901px) { ... }

Mobile-First Approach

/* Base styles (mobile) */ .container { padding: 10px; } /* Tablet styles */ @media (min-width: 600px) { .container { padding: 20px; } } /* Desktop styles */ @media (min-width: 900px) { .container { padding: 30px; } }
