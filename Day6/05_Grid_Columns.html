2. Fraction Units (fr)

The fr unit represents a fraction of the available space.

3. Repeat() Function

4. Auto-Sizing Columns

grid-template-columns: auto auto auto;
Columns will size to fit their content.

5. Minmax() Function

grid-template-columns: minmax(100px, 1fr) minmax(200px, 2fr);
Columns will be at least the first value and at most the second value.
Min 100px, Max 1fr
Min 200px, Max 2fr

6. Responsive Columns

grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
Creates as many 200px columns as will fit, stretching them equally if there's extra space.
